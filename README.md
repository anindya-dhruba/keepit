# KeepIt
Inspired from Google Keep, made with meteorJS, angularJS and &lt;3

----------

### Made With: ###

 - [MeteorJS](https://www.meteor.com/)
 - [AngularJS](https://angularjs.org/)

### Demo: ###
http://keepit.meteor.com


###Installation:###

 - install meteor first (instrucations at [MeteorJS](https://www.meteor.com/))
 - clone the repo
 - create facebook, google and twitter apps, you will need that for signing and logging in the app
 - provide the credentials (appId/clientId/consumerKey, secretKey) in `server/social-config.js`
 - go to project root by running `cd /project_root`
 - build and run the app by running `meteor`
 - browse `http://localhost:3000/`
 - `:)`
